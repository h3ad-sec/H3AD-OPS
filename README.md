# H3AD-OPS

**SOC Operations Hub — Part of [H3AD-SEC](https://h3ad-sec.github.io)**

H3AD-OPS is the SOC operations module of the H3AD-SEC platform. It hosts tools for day-to-day analyst workflows — query libraries, triage aids, and operational tooling.

Live at: [h3ad-sec.github.io/H3AD-OPS](https://h3ad-sec.github.io/H3AD-OPS/)

---

## Tools

### [QUICKTRACE](https://h3ad-sec.github.io/QUICKTRACE/)
A curated library of daily workflow queries organized into four categories: AUTH, NETWORK, ENDPOINT, and CLOUD. Supports KQL, Sigma, and XQL platforms. Filter by category, search by title, and copy to clipboard.
